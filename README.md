# 🎬 Editkaro Portfolio Website

A modern and responsive portfolio website for **Editkaro.in**, a social media marketing and video editing agency.  
Showcases categorized video work, services, and includes lead collection features.

## 🔥 Features

* 🎨 Modern UI with dark theme design  
* 📱 Fully responsive (mobile + desktop)  
* 🎥 Portfolio categorized by video types  
* 📩 Contact form with data storage  
* 📧 Email subscription system  
* ⚡ Fast and optimized performance  
* 🧩 Clean component-based structure  

---

## 🧠 How It Works

1. User visits the website  
2. Explores portfolio categories (Shorts, Ads, Gaming, etc.)  
3. Subscribes via email form  
4. Sends message through contact form  
5. Data is stored (Google Sheets / backend API)
